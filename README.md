# 🔥 스프링클러 펌프 양정 계산기

NFPC 103 / NFTC 103 기준 스프링클러설비 펌프 양정 자동 계산 시스템

## 계산 공식

```
H(m) = H₁ + H₂ + 10m   [식 1-3-3]
```

- **H₁** : 건물높이의 실양정 (토출실양정 + 흡입실양정)
- **H₂** : 배관의 마찰손실수두 (직관 + 부속류 등가길이 기준)
- **10m** : 헤드 방사압력 (고정)

## 내장 데이터

| 표 번호 | 내용 |
|--------|------|
| 표 1-3-8(A) | 일반강관(JIS G 3452) 100m당 마찰손실수두 (헤드 1~30개) |
| 표 1-3-8(B) | 압력배관(JIS G 3454 Sch.40) 100m당 마찰손실수두 (헤드 1~30개) |
| 표 1-3-9 | 관 부속 및 밸브류 상당 직관장 (10개 관경) |

## 주요 기능

- ✅ 흡입측 연성계 mmHg 값 자동 환산 (→ m H₂O)
- ✅ 배관 구간 복수 추가 (관경별 분리 계산)
- ✅ 부속류/밸브류 수량 입력 → 등가길이 자동 계산
- ✅ 일반강관 / 압력배관 선택
- ✅ 헤드개수 → 토출량 자동 연동
- ✅ 계산식 전체 표시 (시험 검토용)
